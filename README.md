# 🛒 Blinkit Sales Insights Dashboard (Power BI)

---

## 📌 Project Overview
This project focuses on analyzing Blinkit’s retail sales data to uncover key business insights related to product performance, customer preferences, and outlet efficiency. The dashboard transforms raw data into interactive visualizations to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance using key KPIs  
- Understand the impact of fat content and item types on revenue  
- Evaluate outlet performance based on size, location, and establishment year  
- Identify trends in customer satisfaction (ratings)  

---

## 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel  
- DAX (Data Analysis Expressions)  

---

## 📊 Key KPIs
- **Total Sales:** $1.20M  
- **Average Sales:** 141  
- **Number of Items:** 8,523  
- **Average Rating:** 3.9  

---

## 📈 Dashboard Features
- Sales analysis by fat content (Low Fat vs Regular)  
- Item type performance comparison  
- Outlet establishment trend over years  
- Sales by outlet size (Small, Medium, High)  
- Geographic sales distribution (Tier 1, Tier 2, Tier 3)  
- Outlet type performance matrix  

---

## 📈 Key Insights
- Low-fat products contribute a significant portion of total sales, indicating strong customer preference  
- Medium-sized outlets generate the highest revenue among all outlet sizes  
- Tier-3 locations show higher overall sales compared to Tier-1 and Tier-2  
- Supermarket Type 1 outlets dominate in both sales and number of items sold  

---

## 🔧 Data Preparation
- Cleaned missing and inconsistent values  
- Standardized categorical fields  
- Created calculated measures using DAX for KPI tracking  

---

## 🚀 Business Recommendations
- Increase inventory for high-performing product categories  
- Focus on expansion in Tier-2 and Tier-3 locations  
- Improve underperforming categories using customer feedback  
- Optimize outlet size strategy for better revenue generation
